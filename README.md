# Business Intelligence — Power BI

A collection of Power BI resources including DAX measures, Power Query (M) scripts, and analytics templates.

## Contents

| File / Folder | Description |
|---|---|
| [`DAX/`](DAX/) | DAX formulas: common measures, KPIs, and calculated columns |
| [`PowerQuery/`](PowerQuery/) | M language scripts for data transformation and loading |
| [`SalesAnalytics/`](SalesAnalytics/) | End-to-end sales analytics example with DAX measures |

## Quick Start

1. Open **Power BI Desktop**.
2. Browse to the relevant folder for the pattern you need.
3. Copy the DAX or M code into the formula bar / Advanced Editor.
4. Adjust table and column names to match your data model.
